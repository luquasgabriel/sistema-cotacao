# Agentes do projeto

Esta pasta contem orientacoes para agentes que forem atuar no Acesso IFAL.
O ponto de entrada na raiz do repositorio e `AGENTS.md`.

## Regras gerais para todos os agentes

- Ler `docs/README.md` antes de propor alteracoes.
- Manter documentacao em portugues.
- Manter codigo e identificadores internos em ingles.
- Preservar a arquitetura Django existente.
- Registrar novas decisoes relevantes na documentacao.
- Evitar implementar regras de negocio diretamente em templates.
